# User Guide

## Registration
1. Navigate to Register page
2. Enter details
3. Select role (Editor recommended)
4. Submit registration

---

## Login
1. Enter registered email and password
2. Login redirects to dashboard

---

## Uploading a Video
1. Click Choose File
2. Select video file
3. Upload starts automatically
4. Processing progress is displayed

---

## Viewing Videos
- Uploaded videos appear in dashboard
- Status shows processing/safe/flagged
- Click play to stream video

---

## Roles
- Viewer: View videos only
- Editor: Upload and manage videos
- Admin: Full access
